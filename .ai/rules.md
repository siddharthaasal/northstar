# Northstar Rules

Never generate mock data.

Prefer server actions over API routes.

Prefer composition over inheritance.

Prefer shadcn components over custom implementations.

Never create duplicate UI primitives.

All forms must use:
- React Hook Form
- Zod

All dates must use:
- date-fns

All database access:
- Drizzle ORM

All icons:
- Lucide

All charts:
- Recharts

Avoid useEffect unless absolutely necessary.

Prefer Server Components.

Prefer optimistic updates.

No inline styles.

No hardcoded colors.

Use semantic Tailwind tokens.

Accessibility required for every component.
