# Northstar Design Principles

Northstar is not a CRUD dashboard.

It is a personal operating system.

The UI should feel:

* Calm
* Fast
* Intentional
* Data-dense
* Minimal

Avoid:

* Marketing-site aesthetics
* Excessive gradients
* Glassmorphism
* Animated backgrounds
* Card overload
* Dribbble-style designs

Inspiration:

* Linear
* Raycast
* Arc Browser
* GitHub
* Things 3
* Apple Health

Design priorities:

1. Fast data entry
2. Daily review
3. Weekly review
4. Historical analysis

Every screen must answer one of these questions:

* What should I do now?
* How am I doing today?
* How have I been doing lately?
* What patterns exist?

The application should be usable entirely from a keyboard.

Dark mode is the primary experience.

Mobile-first but desktop-optimized.

The UI should feel more like a terminal for life metrics than a social app.
